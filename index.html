  • Alunos

    Botão Atualizar Registro

    Ao clicar, atualizará a página.

    Botão Inserir Registro

    Ao clicar, levará até a tela para cadastro de novo aluno.

    Botão Remover Registro

    Ao clicar, deletará da lista o aluno selecionado.

    Botão Exportar Registro

    Ao clicar, gerará um arquivo com todos os alunos cadastrados.

    Botão Salvar Operação

    Ao clicar, salvará as alterações ou inclusões realizadas.

    Botão Cancelar Operação

    Ao clicar, fechará a janela aberta descartando os dados preenchidos.

    Aba Lista

    Visualização dos alunos cadastrados.

    Aba Dados

    Visualização dos dados do aluno selecionado.

    Cadastrando um novo aluno

    Para cadastrar um novo aluno é necessário informar todos os dados solicitados:

    • Nome Completo
    • Situação (registrado, desistente, trabalhando, concluído)
    • Registro (número interno)
    • CPF
    • RG
    • Telefone
    • E-mail
    • Data de nascimento
    • Gênero
    • Cor da pele
    • Raça predominante
    • Nome da mãe
    • Nome do pai
    • Endereço
    • Informações de saúde
    • Informações de educação

    Após preencher todos os dados, clicar no botão salvar operação - para salvar - ou botão cancelar opereção para cancelar. Ao salvar, o aluno será inserido na lista de alunos cadastrados.

    Cadastrando uma nova turma

    Para cadastrar uma nova turma é necessário informar todos os dados solicitados:

    • Descrição da turma
    • Controle (número interno)
    • Vagas
    • Abertura (data de início da turma)
    • Encerramento (data de término da turma)
    • Selecione o instrutor da turma

    Após preencher todos os dados, clicar no botão salvar operação - para salvar - ou botão cancelar opereção para cancelar. Ao salvar, a turma será inserida na lista de turmas cadastradas.
